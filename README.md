# Tutor Appointment Booking System

## Project Overview
This backend application is built using Node.js and TypeScript following Clean Architecture. It allows users to book tutor appointments and admins to manage them.

## Features
- JWT Authentication
- Role-based Authorization
- Tutor APIs
- Appointment booking system
- Admin approval/decline/edit
- Logging
- Unit testing with Jest

## Architecture
- Domain Layer
- Application Layer
- Infrastructure Layer
- Presentation Layer

## Testing
Achieved more than 80% coverage in all categories.

## Run Project
npm install
npm run dev

## Run Tests
npm run test
npm run test:coverage
